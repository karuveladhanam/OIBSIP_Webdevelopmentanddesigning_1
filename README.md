 🍔 Foodie Website
A modern and responsive food ordering website built using HTML, CSS, and JavaScript. This project allows users to explore popular dishes, search for food items, and enjoy a clean, interactive user interface.

✨ Features
🍽️ Hero Section – Attractive landing section with call-to-action
🔍 Food Search – Dynamically filter food items
📋 Menu Section – Displays popular dishes
⚡ Smooth Scrolling – Easy navigation across sections
📱 Responsive Design – Works on mobile and desktop
🎨 Modern UI – Clean and professional layout

🛠️ Technologies Used
HTML5
CSS3
JavaScript

🚀 How to Run the Project
Clone the repository:
Open the folder
Run the project:
Double-click **index.html**
OR
Open with any browser

🎯 Functionality
Click **Order Now** → Smooth scroll to menu section
Type in search box → Filters food items instantly
Hover on cards → Interactive hover effect
Click navigation links → Moves to respective sections
